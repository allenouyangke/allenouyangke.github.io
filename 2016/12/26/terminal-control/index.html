终端操作快捷键

发表于 2016-12-26 | 分类于 Shell | | 本文总阅读量 次

本篇概括
  对于经常使用命令行，掌握一些常用的快捷键是非常有必要的。它能帮助你提升工作的效率，逐渐从鼠标的操作转变成键盘操作。渐渐的，你就会感受到命令行的美妙之处！

常用操作

  • Ctrl + L
    用于清理终端的内容，就是清屏的作用。其实clear命令也有同样效果，但是你不觉得Ctrl + L 的按键是比输入clear这五个字母更快速吗？

  • Ctrl + D
    给终端传递EOF （End Of File，文件结束符），在运行程序时很有用，有些程序我们需要在接收到EOF输入时结束，那么这个快捷键就可以派上用场了。如果你在命令行提示符后什么也不输入的情况下直接按下这组快捷键，那么就会关闭当前的终端。

  • Shift + PgUp
    用于向上滚屏，与鼠标的滚轮向上滚屏是一个效果，但岂不是很方便吗？

  • Shift + PgDn
    用于向下滚屏，与鼠标的滚轮向下滚屏是一个效果。

长命令操作

  • Ctrl + A
    光标跳到一行命令的开头。一般来说，Home键有相同的效果。

  • Ctrl + E
    光标跳到一行命令的结尾。一般来说，End键有相同的效果。

  • Ctrl + U
    删除所有在光标左侧的命令字符。

  • Ctrl + K
    删除所有在光标右侧的命令字符。

  • Ctrl + W
    删除光标左侧的一个“单词”，这里的“单词”指的是用空格隔开的一个字符串。例如 -a 就是一个“单词”。

  • Ctrl + Y
    粘贴用Ctrl + U， Ctrl + K或Ctrl + W“删除”的字符串，有点像“剪切-粘贴”。
