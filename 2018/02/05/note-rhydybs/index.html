《如何有效阅读一本书：超实用笔记读书法》笔记

发表于 2018-02-05 | 分类于 Read | | 本文总阅读量 次

  本书的作用：指导你用好笔记，经过规划、梳理，学习怎样选书、怎样读书和怎样把书中的内容刻在脑海里并加以活用，然后把这些做法转化为实用、可持续、属于自己的技巧。

前言

  • “一味读下去”等于没有读

  • 这本书讲了什么内容？
    你最喜欢书的哪一部分？
    本书对你有什么影响？
    它的优点又在哪里？

  • 从现在开始，认真选择每一本能满足自己需求的书，然后真挚地与每一本书对话，努力从书中学到知识吧。

  • 把书中的知识转化为“智慧”

  • 通过做笔记吸取书中营养

  • 要买什么书？
    要注意书中的哪个部分？
    想到什么？

第一章 用笔记管理读书生活

人与书关系的改变者

  • 读书不再是“随随便便”地读，而是带着明确的目的、充满主动性地去
    真正消化书中的信息，使之称为属于自己的东
    深入理解书中的要点或思想，并随时拿来参考

  • 选书、购书、读书和活用

    • 如何选书：怎样选择自己真正想读的书
    • 如何购书：怎样买到对自己真正有益的书
    • 如何读书：怎样加深理解、深入思考
    • 如何活用：怎样运用从书中获得的知识

让笔记成为读书时的好伙伴

  • 随想笔记

    1. 记下平时的感悟，与日后的选书息息相关。
    2. 不需要对这些信息进行分类整理，只要按照时间顺序将他们排列在笔记本上就够了。

  • 购书清单

    1. 找真正想读的书。
    2. 主动地选择书籍。

  • 各种报道的简报

    • 尽情收录感兴趣的内容。

  • 读书笔记

    1. 将读后感转变成文字保存下来。
    2. 笔记本即将成为你读书生活的好搭档。
    3. 我们不可能预测出“怎样才能遇到想读的书”，所以不要去分类，直接把信息都罗列在笔记本上反而更好。

只要坚持就一定会有效

  • “好记信不如烂笔头”
  • 将来总有天你能用到的知识储备。
  • 把灵感记录在纸上，让思考书面化，让灵感脱离之前的抽象状态，会更容易将其理解消化，也能促使你把灵感用作创意材料，想用的时候就能信手拈来。

通过笔记养成良好的读书习惯

三个窍门

  1. 养成习惯
  2. 放大读书笔记的作用
  3. 创造别具一格的记录方式

完善购书、读书和活用方式

  • 更好的读书体验便可以成就全新的自己。

  • 一般情况下，读一本书需要经过三个步骤：

    1. 购书：在实体店或网店购买喜欢的书
    2. 读书：普通的读书过程
    3. 归档：读后将书归类的过程

  • 本书情况下，读书分为五个阶段：

    1. 选书
    2. 购书
    3. 读书
    4. 记录

    5. 活用

      其实在读这本书之前，我也已经渐渐的摸索到这种方式了，曾经我也是很苦难为何自己毕业后就看书很难入脑了。更多的原因是时间的碎片化和内心的浮躁，导致以往的读书方式已经不适用了

将读书过程结构化

   对于这个概念，我是这样认为的。就是将读书的五个阶段编程你的阅读习惯，唯有成为了习惯，才不至于觉得阅读那么变扭。

把读书相关信息记入同一本笔记

  • 使用同一本笔记本记录所有信息的重要原因——“信息一元化”

    • 一元化的优点
      1. 简单易懂，人人都能做到
      2. 更容易坚持
      3. 自由度高，可自行设置
      4. 信息“存在”与同一个地方
      5. 直接记录、参考，不易混淆
      6. 因为“杂乱无章”，所以更便于活用

  • 在一本写什么都可以的笔记本里，增加关于读书的记录
    就目前我个人来说，因为使用了macbook，基本这就成为了我的笔记本了，我开始将所有的读书笔记、生活随笔、人生感悟都写入到了macbook中。

  • 《信息管理术》
    先记录下来，有空去看看。

  • 只要在这里找，就一定能找到；在这里找不到的话，那就是没有。

笔记本一元化技巧

  • 五要点：

    1. 可以书写任何内容
    2. 按时间顺序书写（只要写上就好）
    3. 附上日期
    4. 灵活运用速记或略记
    5. 可以粘贴

  • 一日一记的日记式写法，会因为偶尔写不出内容而出现空白，这样就会使人产生懈怠感。
    这两年经常提醒自己要写日记，最初有经常写，但工作一忙就忘记了。现在通过这种方式，或许会好很多。更多时候应该是我的强迫症太过严重，导致我的笔记都没能坚持下来。

  • 日期标记
    标注年号和日期有多种多样的方式，为了避免混乱，应该尽量统一使用统一方式标记。

  • 速记/略记

    • 速记：
      速记的窍门就是用较大的字号书写。或者是写一行空一行的方式，笔记本用完了就换嘛。
    • 简写：
      为了更方便辨识，可以将冗长的表述缩写为字数更加少的短语。
    • 略记：
      比较常用就是首字母进行缩写。
      这种方式试过，但是个人用不习惯。

  • 可以粘贴
    如果没有电脑我可能会使用纸质笔记，但是开始使用macbook开始完全抛弃了纸质笔记本了。加上现在拍照、截图等十分方便，更多时候使用可截图保存即可，google的图床也没有限制容量，只要有VPN爱怎么传就怎么传。

第二章 用购书清单指名购书

购书清单是一份无形的财富

  • 笔记读书法的第一步，就是选购真正想读、应该读的书，也就是五步中的“选书”和“购书”。

  • 购书清单
    三线索：书名、作者名、出版社
    实现指名购买

书名 作者 出版社 参考价 备注

  • 列读书清单的理由之一——培养带着目的去读书的目的意识

  • 列出读书清单→在书店/商城确认→购买
    遵循这样的步骤入手书籍，就不得不意识到自己“为什么要读这本书”了。通过在清单上书写这一动作，可以确认自己此次读书的目的。

  • 带着理由去读书的意识，可会影响阅读速度

通过记录掌握选书的主动权

  • 列读书清单理由之二——能帮助自己找到真正想读的书

  • 平日买书的根本原因——冲动购买
    站在书架前，很容易失去判断力，无法冷静选购。

  • 列读书清单——为了促进主观购书的矫正装置

  • 列读书清单——看清自己真正的读书需求
    只有从日常生活中取材，才能列出反映自己真正需求的购书清单。

在实体店购书印象更深刻

对于个人而言，我更加倾向于购买电子书和网上购书，虽然不及在实体店购买印象深刻，但起码是我需要使用的书籍。我觉得购买只是获得书籍的一种方式，购书的真正目的自己必须考虑清楚。一如我以往买笔记本一样，到了文具货架前，看到笔记本总是有购买的冲动，感觉自己能用的上；而实际上，我的笔记本堆了一整个书架。

列购书清单的诀窍

  • 购书清单：就是一份有别于笔记本的专用表格。

  • 只要按照“书名”、“作者”、“出版社”三个要素检索购书清单上的书籍，一般吨那个找到目标书籍。

  • “首字母缩写法”
    由于实体书店比较少，而且标识各异，所以实体书店中基本很少去，这种标识的方法也基本用不上。

在线清单的优缺点

  • 亚马逊“心愿单”
    我的书单基本是写在了wiz笔记或者博客上，最主要是方便我随时随地的查看和调用。

指名购买的威力

  • 购书清单上的书籍可以分为两类：

    • 不买也行的书：划掉
    • 必须购买的书：已购

  • 一般来说，去书店的需求分为三种：

    1. 想读书，但还没决定好读什么
    2. 寻找某一特定领域、系列或作者的书
    3. 想读某本特定的书

在日常生活中手机线索

  • 选择图书，重要的是看平时手机的信息，要经常竖起敏感的天线。平时提高了敏感度，就算没有特意去找，也能在日常生活里收集很多有意思的信息。这样积累下去，想读的书要多少有多少，根本不需要去网上搜索。

  • 选择书的方式可以理解为：主动/被动声呐，最先贯彻的应该是“被动声呐”。

  • 不管你对一件事有多关注，多感兴趣，如果不马上记录下来，很容易就会忘记。所以计算是一句话也好，把主题或关键字写下来，说不定就能让你邂逅好书。

让清单和笔记本联动起来

  • 不要放走任何与读书有关的信息——“读书信息”不只是读书笔记和购书清单那么简单的事。

  • 日常笔记或报纸简报→购书清单→读书→重读笔记或剪报

找到能帮助自己找书的“枢纽书”

  • 书里推荐书

  • 不断引领我们找到其他书的书称为“枢纽书”。

了解报纸的出版习惯

  • 判断报社的编辑方针
    报纸现在基本没有看了，也没有这样的时间。更多的信息来源是通过互联网，看到有好的文章和文字，都会在第一时间里，使用wiz的剪辑器将其剪辑下来，保存到个人笔记上；好的文章会先添加到阅读栏中，等有空了再细细品读。

认真研读报刊书评

  • 报纸、杂志等纸媒体具有易保存性，其一大好处正是可以无限“熟读”。

  • 书评可以帮助理解。

  • 书评是写给没有读过某本书的人的文章，所以会先说这本书的概要，之后的内容就是评论者的感想了。所以只要读懂前半部分，就能整理出本书的主要框架或故事概要。

第三章 用笔记把读过的书变为精神财富

笔记让读书成果一目了然

  • 重点在“读书”和“记录”

  • 一切都应该以“能坚持下去”为先

  • 读书笔记的作用：
    “图书都是有重点的，没必要从头到尾详读。重要的是不管篇幅多少，都要下点功夫让自己对读过的书有印象。即使只记下了这本书的作者、书名、阅读时的情景或是做些摘抄都好，要坚持写读书笔记，这是最有效的读书方法。”

读书笔记改变读书方法

  • “你为别人讲解书中的内容时，才会真正理解它。”
    把读书笔记作为目标去读书，得到的效果也是一样的。当你以思想输出位前提去读书时，思想输入的质量也会有所提升，而且亲手写文章的好处比口头叙述要多。

  • “摘抄+感想”
    目前我就是采取的这种读书方式。

  • 读书的目的从“读完就好”转变为“写读书笔记”；读书的重心也自然发生变化：不再是“因为读了书而写读书笔记”，而是“为了写读书笔记而读书”。

从一句话开始

  • 读书的窍门：要彻底专注于“对自己很重要的事情”

  • 写在读书笔记上的感想可以只是一句话。

  • 记录感想的时候，不要写一些揣测的或者外来的想法，而是要写下自己的心声，也就是感受和想法，这样会让读书笔记更有内涵。哪怕是一句话，只要写的是发自内心的语句，那么这次读书笔记就是有意义的。

  • 同“整理自己的想法”

无法坚持做笔记的原因

  • 读书笔记就像日志一样，一旦停止就很难恢复，如果搁置了很久，重新开始做笔记时就会产生强烈的抵触感。

  • 不要为了坚持写读书笔记而走形式主义。不管采用什么方式，都要踏实地写笔记，并真正理解书的内容。

巩固印象的笔记

  • 在写读书笔记这个前提下，读书方法会发生变化，你会进入主动寻找书中“打动人心的内容”的工序。

  • 寻读/跳读/泛读
    一边寻找，一边划分好“部分”，按照不同的方法阅读。

  • 读书也可以是一项“对重要内容进行提炼的工序”

  • 读书、画重点、写读书笔记

做记号的三个步骤

  • 筛选程序
    • 通读
      一边阅读，一边把觉得有价值的那一页折角
    • 重读
      读完一遍之后，再把折角的几页重新读一遍。如果仍然觉得很好，就把那一页另外一角也折起来
    • 标记
      再重新读一遍折起上下两角的几页，如果第三次阅读后仍然觉得值得一读，就用笔在上面做记号。

利用书签边读边写

  • 准备三枚书签（通读书签、重读书签、标记书签）
  • 在“通读”中断时，夹好“通读书签”
  • 通读到某一部分时，开始“重读”程序，记得在中断时夹好“重读书签”
  • 中断“重读”程序以后，开始“标记”程序，中断时夹好“标记书签”
  • 三枚书签都夹在书里以后，就可以准备读书笔记了

巧用标记区分对象

  • 画线方式：
    • 普通直线：用于“重要”“客观上很重要““暂时先画出”的情况
    • 波浪线：用于“非常重要”“主观觉得值得重视”“想记住这里，以后加以运用”的情况
    • 圆圈：用来圈出重要的专有名词、关键词、关键句等需要吸引眼球的内容

“葱鮪火锅式”读书笔记

  • 三要素

    1. 写读书笔记的日期
    2. 书名
    3. 作者名

  • 二要素

    1. 对自己来说重要的内容（摘抄）
    2. 自己对这篇文章的感想（评论）

  • 标记方式

    • 圆圈：后面是摘抄。不要省略，保持原味。
    • 五角星：后面是自己的评论和感悟。

  • “摘抄对自己来说很重要的文章”和“写主观评论”上即可。

通过摘抄促进对数的消化

  • 摘抄的第一个好处——促进、加深对书的记忆

  • 总结要点比抄写原文还需要动脑，更容易让人产生疲劳心理，所以还是抄写原文的好。

  • 摘抄的第二个好处——加深理解

寻找最具有代表性的语句

  • 一定要摘抄让自己心动的语句。

  • 带着主观的心情去读书，是学者和编者都没有的，只属于普通读者的特权，应该放心享受才是。

  • 摘抄的内容，应该让人感觉“这么一说确实有道理“的内容。

  • 如果一篇文章颠覆了你之前的想法使你的认识发生动摇，在抄写和重读的过程中仍然会让人信服或是感觉震撼，这种文章才是值得摘抄的。

  • 筛选关键句的工序本身也是一种重读。

  • 最能代表这次读书体验的语句就是最理想的状态。

  • 放置一段时间后仍然经得起考验的文章，才真有价值。

激发思想的火花

  • 自己当下的想法才是最重要的，摘抄这一段文章的原因和自己当时的想法才是日后值得参考的贵重资源。
